# 🌸 Pokemon TCG Wishlist

A beautiful, responsive web application for managing your Pokemon Trading Card Game wishlist with AI-powered search capabilities.

## ✨ Features

- 🎯 **Smart Card Search**: AI chatbot helps you find specific Pokemon cards
- 📱 **Mobile Responsive**: Works perfectly on both desktop and mobile
- 🎨 **Beautiful UI**: Pastel pink theme with clean, modern design
- 💾 **User Accounts**: Secure login/registration with personalized data
- 📋 **Wishlist Management**: Add cards to your wishlist and mark cards you already own
- 🔍 **Advanced Filtering**: Filter by rarity, type, and more
- 💰 **Card Pricing**: View current TCG Player prices
- ⚡ **Fast Loading**: Optimized with instant initial load and background updates
- 🃏 **Comprehensive Database**: Access to thousands of Pokemon and Trainer cards

## 🛠️ Technology Stack

- **Frontend**: React Native with Expo
- **Styling**: Custom StyleSheet with responsive design
- **Data**: Pokemon TCG API + Local caching
- **Authentication**: Custom auth service with AsyncStorage
- **Deployment**: Vercel

## 🏃‍♂️ Running Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/pokemon-trainer-wishlist.git
   cd pokemon-trainer-wishlist
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run web
   ```

4. **Open your browser**
   Navigate to `http://localhost:8081`

## 📱 Mobile App

This app is built with Expo and can also run as a mobile app:

```bash
# Install Expo Go on your phone
npm start
# Scan the QR code with Expo Go
```

## 🎮 How to Use

1. **Sign Up/Login**: Create an account to save your data
2. **Browse Cards**: Explore Pokemon and Trainer cards
3. **Search**: Use the AI chatbot to find specific cards
4. **Add to Wishlist**: Click the heart icon on cards you want
5. **Mark as Owned**: Click "Already Have" for cards you own
6. **Filter**: Use advanced filters to find cards by rarity, type, etc.
